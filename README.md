# Jacquot Digital Solutions

Production-ready React + Vite website for Jacquot Digital Solutions, a custom business software company in Walkerton, Ontario.

## What is included

- React + Vite application structure
- Reusable section and content components
- Responsive, accessible layouts
- Custom generated hero asset stored in `public/assets`
- SEO metadata, Open Graph tags, structured data, robots file, and sitemap
- Modern CSS with performance-friendly animations
- Contact form markup ready to connect to a backend or form provider

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Project structure

```text
src/
  components/        Reusable page sections
  data/              Site copy and content arrays
  App.jsx            Page composition
  main.jsx           React entry point
  styles.css         Global design system and responsive styling
public/
  assets/            Static visual assets
  favicon.svg
  robots.txt
  sitemap.xml
```
